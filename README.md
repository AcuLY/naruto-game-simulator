# Naruto-Game-Simulator

## 游戏规则

**初始状态**：`1` 条生命，生命值 `2` 点，生命上限 `4` 点，查克拉 `0` 点

**回合执行顺序**：
- 玩家选择招式
- 玩家选择目标
- 技能执行：
- **看透**玩家预选择招式和目标

**招式执行顺序**：先攻 $\rightarrow$ 空间 $\rightarrow$ 状态 $\rightarrow$ 主动 $\rightarrow$ 篡改

**球**：主动招式的攻击实体，由**源**指向**目标**

- **束缚**：下一回合无法行动
- **看透**：立即选择下一回合的行动，并告知所有玩家
- **吸血**：如果成功造成伤害，则增加等量的生命值
- **封穴**：下一回合只能使用消耗为 `0` 的行动
- **影分身**：下一回合的行动执行两次（如果查克拉足够，否则无效果）
- **六道模式**：持续 `2` 回合，模式内可使用六道技能
- **魂吸**：将目标查克拉清零，并获取等量的查克拉，多个魂吸对一个目标时，每个源都能获得目标原本的查克拉量
- **秽土**：如果目标死亡，则下回合以**初始状态**复活；如果目标存活，则增加 `1` 条命，最多 `2` 条，超出无效，新的生命为**初始状态**，但 `2` 条命的查克拉量共享
- **封尽**：无法再进入**秽土**状态
- **空间转移**：游戏存在两个空间（现实空间和神威空间），初始均为现实空间，招式只能选择同一个空间内的目标，当且仅当进行空间转移时能切换空间，死亡、秽土等状态均不会改变空间
